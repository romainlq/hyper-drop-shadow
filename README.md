# hyper-drop-shadow

Simple plugin that disable the drop shadow on macOs.

## Install

### Using hyper

To install, either use
```
hyper i hyper-drop-shadow
```

### Manually

To install, just edit your `~/.hyper.js` and add `"hyper-drop-shadow"` to `plugins`:
```
plugins: [
  "hyper-drop-shadow"
],
```
